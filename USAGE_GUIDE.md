# LaTeX Clipboard Converter - 사용 가이드

## 빠른 시작

### 1. 앱 실행

앱을 실행하면 메뉴바에 📐 아이콘이 나타납니다.

### 2. API 키 설정

1. 메뉴바 아이콘 클릭
2. "Settings..." 선택
3. Claude API 키 입력
4. "Test" 버튼으로 키 확인
5. "Save" 클릭

### 3. 사용하기

1. LaTeX 수식이 포함된 이미지를 스크린샷으로 캡처 (⌘⇧4)
2. 자동으로 LaTeX 코드로 변환됨
3. 원하는 곳에 붙여넣기 (⌘V)

## 상세 가이드

### Claude API 키 발급

1. https://console.anthropic.com/ 접속
2. 계정 생성 또는 로그인
3. API Keys 메뉴에서 새 키 생성
4. 키를 복사하여 앱 설정에 입력

### 메뉴 설명

#### ✓ Enabled / ☐ Disabled
- 클립보드 모니터링 활성화/비활성화
- 비활성화하면 배터리 절약

#### ☐ Launch at Login
- 로그인 시 자동 실행
- macOS 13 이상 필요

#### Settings...
- API 키 설정
- 폴링 간격 조정 (0.1~2.0초)
- 기본값: 0.5초

### 사용 팁

#### 최적의 스크린샷

1. **고해상도**: 선명한 이미지일수록 정확도 향상
2. **수식만 캡처**: 주변 텍스트 제외
3. **적절한 크기**: 너무 작거나 크지 않게

#### 배터리 절약

1. 사용하지 않을 때 비활성화
2. 폴링 간격을 1초 이상으로 설정
3. 필요할 때만 활성화

#### 정확도 향상

1. 표준 LaTeX 표기법 사용
2. 깨끗한 배경
3. 충분한 대비

### 문제 해결

#### 변환이 안 됨

**증상**: 이미지를 캡처해도 변환되지 않음

**해결 방법**:
1. 메뉴에서 "✓ Enabled" 확인
2. 시스템 환경설정 > 보안 및 개인 정보 보호 > 손쉬운 사용에서 앱 권한 확인
3. 폴링 간격을 0.5초로 설정
4. 앱 재시작

#### API 키 오류

**증상**: "Invalid API key" 에러

**해결 방법**:
1. API 키를 다시 복사하여 입력
2. 공백이나 줄바꿈 제거
3. Claude 콘솔에서 키가 활성화되어 있는지 확인
4. API 크레딧 잔액 확인

#### 변환이 부정확함

**증상**: LaTeX 코드가 원본과 다름

**해결 방법**:
1. 더 선명한 이미지로 다시 시도
2. 수식만 캡처 (주변 텍스트 제외)
3. 표준 LaTeX 표기법 사용
4. 복잡한 수식은 여러 부분으로 나누어 캡처

#### 느린 변환

**증상**: 변환에 시간이 오래 걸림

**원인**:
- 인터넷 연결 속도
- Claude API 서버 응답 시간
- 이미지 크기

**해결 방법**:
1. 인터넷 연결 확인
2. 이미지 크기 줄이기
3. 10초 이상 걸리면 타임아웃 (재시도)

### 고급 설정

#### 폴링 간격 조정

**0.1초**: 
- 장점: 즉각적인 반응
- 단점: 배터리 소모 증가

**0.5초 (기본값)**:
- 균형잡힌 설정
- 대부분의 사용자에게 권장

**1.0초 이상**:
- 장점: 배터리 절약
- 단점: 약간의 지연

#### Launch at Login

**활성화 시**:
- 로그인 시 자동 실행
- 항상 사용 가능

**비활성화 시**:
- 수동 실행 필요
- 필요할 때만 사용

### 사용 예시

#### 예시 1: 논문 작성

1. PDF 논문에서 수식 스크린샷
2. 자동으로 LaTeX 코드 생성
3. Overleaf나 LaTeX 에디터에 붙여넣기

#### 예시 2: 강의 노트

1. 칠판/슬라이드의 수식 사진 촬영
2. 사진을 클립보드에 복사
3. LaTeX 코드로 변환
4. 노트 앱에 붙여넣기

#### 예시 3: 수학 문제 풀이

1. 문제집의 수식 캡처
2. LaTeX 코드 획득
3. 온라인 수식 편집기에서 수정
4. 과제 제출

### API 비용 관리

#### 예상 비용

- Claude 3.5 Sonnet: 이미지당 약 $0.003
- 1,000개 이미지: 약 $3
- 월 100개 사용 시: 약 $0.30

#### 비용 절약 팁

1. 필요한 수식만 변환
2. 비슷한 수식은 한 번만 변환 후 수정
3. 간단한 수식은 직접 입력
4. API 사용량 모니터링

### 키보드 단축키

현재 버전에는 커스텀 단축키가 없지만, macOS 기본 단축키 사용 가능:

- **⌘⇧4**: 스크린샷 캡처
- **⌘V**: 붙여넣기
- **⌘Q**: 앱 종료

### 지원되는 LaTeX 요소

#### 완벽 지원
- 기본 수식: `x^2`, `\frac{a}{b}`
- 그리스 문자: `\alpha`, `\beta`
- 적분/미분: `\int`, `\frac{d}{dx}`
- 행렬: `\begin{matrix}...\end{matrix}`
- 합/곱: `\sum`, `\prod`

#### 제한적 지원
- 복잡한 다이어그램
- 손글씨
- 매우 작은 글씨
- 흐릿한 이미지

- 로컬에 저장되지 않음
- API 키는 UserDefaults에 저장
